# task_helper

このアプリは 学校の課題の提出を管理するアプリです。

## 機能
- クラスルームから課題情報を取得して、課題の提出の有無を確認
- メンタル状況（睡眠時間の記録、気分の記録）

## 技術スタック
- フレームワーク: Flutter
- 言語: Dart, Kotlin
- DB: SQLite
- API: Google Classroom
